# PVote
##A simple python web vote page use BaseHttpServer 
##简单web版投票系统

###Install(need [WShell](https://github.com/QGB/Wshell))
```
ginit PVote
cd qgb
ginit qpsu
```
###Run
```
Start-PVote.bat
```
